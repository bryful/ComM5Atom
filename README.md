# ComM5Atom

M5Stack AtomS3とシリアル通信を行うためのテスト用のC#WindowsDesktopアプリ<br>
.netframework4.8で作っています。<br>
<br>


[AtomS3_Display](https://github.com/bryful/AtomS3_Display)のリポジトリがM3Stackのコードになります。

## Dependency
Visual studio 2022 C#
